Base Modules

The following modules are included in the standard distribution and can be loaded at run-time with mod.load() in Fluid or LoadModule() in C/C++.

Use the navigation bar on the right to peruse the available functionality of the selected module.

Beginners should start with the Core module, which includes the bulk of Parasol's functionality.

GetDisplayMode()

Retrieve information of a display mode.

APTR xrGetDisplayMode(LONG Index)
ParameterDescription
IndexIndex of the display mode to retrieve.

Description

This function returns the width, height and depth of a given display mode.

Result

An xrMode structure is returned or NULL on failure.

GetDisplayTotal()

Returns the total number of display modes.

LONG xrGetDisplayTotal()

Description

This function returns the total number of known display modes.

Result

Returns the total number of known display modes.

SetDisplayMode()

Change the width and height of the display.

ERROR xrSetDisplayMode(LONG * Width, LONG * Height)
ParameterDescription
WidthThe required width of the display.
HeightThe required height of the display.

Description

This function changes the width and height of the display to that indicated by the Width and Height parameters. If the requested size does not match a known mode, the closest matching mode will be chosen.

Error Codes

OkayOperation successful.
